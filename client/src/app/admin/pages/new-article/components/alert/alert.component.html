При написанні статті дотримуйтесь наступних правил:
  1. Кожен абзац статті огортайте в тег <p style="text-indent:30px"></p>
  2. Для виділення тексту жирним шрифтом використовуйте тег <span style="font-weight:600"></span>
  3. Для виділення тексту курсивом використовуйте тег <span style="font-style:italic"></span>
  4. Для виділення тексту жирним курсивом використовуйте тег <span style="font-style:italic;font-weight:600"></span>
  5. Для набору математичних формул використовується синтаксис мови розмітки документів LaTeX
